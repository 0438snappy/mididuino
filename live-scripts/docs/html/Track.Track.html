arm Arm the track for recording. Not available for Master- and Send Tracks. property
can_be_armed return True, if this Track has a valid arm property. Not all tracks can be armed (for example return Tracks or the Master Tracks). property
canonical_parent Get the canonical parent of the track. property
clip_slots const access to the list of clipslots (see class AClipSlot) for this track. The list will be empty for the master and sendtracks. property
current_input_routing Get/Set the name of the current active input routing. When setting a new routing, the new routing must be one of the available ones. property
current_input_sub_routing Get/Set the current active input sub routing. When setting a new routing, the new routing must be one of the available ones. property
current_monitoring_state Get/Set the track's current monitoring state. property
current_output_routing Get/Set the current active output routing. When setting a new routing, the new routing must be one of the available ones. property
current_output_sub_routing Get/Set the current active output sub routing. When setting a new routing, the new routing must be one of the available ones. property
devices Return const access to all available Devices that are present in the Tracks Devicechain. This tuple will also include the 'mixer_device' that every Track always has. property
fired_slot_index const access to the index of the fired (and thus blinking) clipslot in this track. This index is -1 if no slot is fired and -2 if the track's stop button has been fired. property
fold_state Get/Set whether the track is folded or not. Only available if is_foldable is True. property
has_audio_input return True, if this Track can be feed with an Audio signal. This is true for all Audio Tracks. property
has_audio_output return True, if this Track sends out an Audio signal. This is true for all Audio Tracks, and MIDI tracks with an Instrument. property
has_midi_input return True, if this Track can be feed with an Audio signal. This is true for all MIDI Tracks. property
has_midi_output return True, if this Track sends out MIDI events. This is true for all MIDI Tracks with no Instruments. property
input_meter_left Momentary value of left input channel meter, 0.0 to 1.0. For Audio Tracks only. property
input_meter_level Return the MIDI or Audio meter value of the Tracks input, depending on the of the Track input. Meter values (MIDI or Audio) are always scaled from 0.0 to 1.0. property
input_meter_right Momentary value of right input channel meter, 0.0 to 1.0. For Audio Tracks only. property
input_routings Const access to the list of available input routings. property
input_sub_routings Return a list of all available input sub routings. property
is_foldable return True if the track can be (un)folded to hide/reveal contained tracks. property
is_part_of_selection return False if the track is not selected. property
is_visible return False if the track is hidden within a folded group track. property
mixer_device Return access to the special Device that every Track has: This Device contains the Volume, Pan, Sendamounts, and Crossfade assignment parameters. property
mute Mute/unmute the track. property
name Read/write access to the name of the Track, as visible in the track header. property
output_meter_left Momentary value of left output channel meter, 0.0 to 1.0. For tracks with audio output only. property
output_meter_level Return the MIDI or Audio meter value of the Track output (behind the mixer_device), depending on the of the Track input, this can be a MIDI or Audio meter. Meter values (MIDI or Audio) are always scaled from 0.0 to 1.0. property
output_meter_right Momentary value of right output channel meter, 0.0 to 1.0. For tracks with audio output only. property
output_routings Const access to the list of all available output routings. property
output_sub_routings Return a list of all available output sub routings. property
playing_slot_index const access to the index of the currently playing clip in the track. Will be -1 when no clip is playing. property
solo Get/Set the solo status of the track. Note that this will not disable the solo state of any other track. If you want exclusive solo, you have to disable the solo state of the other Tracks manually. property
view Representing the view aspects of a Track. property
